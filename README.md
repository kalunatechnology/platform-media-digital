# Cara install aplikasi

## Requirement

- PHP 8.0 - 8.2

## cara Install 
`git clone https://github.com/kalunatechnology/quick_count.git`
`composer install`
`php artisan storage:link`
`npm install` 
`npm run dev` 
`ubah file env.example menjadi .env` 
`sesuaikan konfigurasi env`
