# Cara install aplikasi

## Requirement

- PHP 8.0 - 8.2
- MYSQL

## cara Install 
`git clone https://github.com/kalunatechnology/platform-media-digital.git`
`composer install`
`php artisan storage:link`
`npm install` 
`npm run dev` 
`ubah file env.example menjadi .env` 
`sesuaikan konfigurasi env`
